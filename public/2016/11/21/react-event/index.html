React学习之基础篇（四）———— 事件机制

发表于 2016-11-21 | 分类于 React | | 阅读次数

学习React中的事件机制，理解语法及本质。

React 处理事件的不同之处

  • 事件采用驼峰命名

  • 无法直接使用return false;组织事件默认行为

  • React 中的事件都是虚拟的，但兼容性是大大的，不必使用addEventListner进行事件监听，相反，只需在初始渲染时提供一个listener即可。当使用ES6 class，进行组件定义时，可以采用handler event的方法，届时调用即可
